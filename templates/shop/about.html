About BubuKappa

Welcome to BubuKappa, a cozy online sanctuary for readers, specializing in manga, light novels, and classic literature. This platform is a fully featured personal graduation project designed to demonstrate clean web architecture, robust database relations, and dynamic user interfaces.

Under the conceptual guidance of our mascot, the capybara, this bookstore features complex database querying, deep catalog filtering, asynchronous shopping cart updates, and personalized user session tracking. Every single system has been constructed from scratch using Python, Django, and modern JavaScript.

Thank you for taking the time to explore this development showcase. Feel free to interact with all the features.
