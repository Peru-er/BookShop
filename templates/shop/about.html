About BubuKappa

Welcome to BubuKappa! This platform is a course project designed to simulate a modern, high-performance online bookstore specializing in books, manga and light novels.

The goal of this project is to implement robust database structures, deep filtering mechanisms, asynchronous cart workflows (AJAX), and personalized user sessions. Every feature you interact with—from search queries to the simulated stock waitlists—has been developed from scratch using Django and Bootstrap 5.

Thank you for visiting my work. Feel free to test out all functionalities.
