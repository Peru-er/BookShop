
# BubuKappa

BubuKappa — навчальний проєкт інтернет-магазину, створений на Django.  
Проєкт реалізує базову e-commerce платформу для продажу книг і манги.

## Основна ідея

Магазин призначений для продажу книг, манги та пов’язаних видань.  
Підтримуються категорії товарів, автори, серії, жанри та зображення товарів.

## Етапи розробки

### Етап 1 — Базова структура
- Налаштування проєкту
- Створення моделей:
  - Category
  - Product
  - ProductImage
- Налаштування Django Admin:
  - фільтри
  - inline-редагування зображень
- Реалізація каталогу товарів:
  - список товарів
  - сторінка деталей товару

### Етап 2 — Користувачі та кошик
- Реєстрація та авторизація користувачів
- Профілі користувачів
- Пошук і фільтрація товарів
- Сесійний кошик

### Етап 3 — Оформлення замовлення
- Адреса доставки
- Створення замовлення
- Історія замовлень
- Email-сповіщення

### Етап 4 — Оптимізація та розгортання
- Кешування (Redis)
- Базова безпека (CSRF, XSS, SQL Injection)
- Docker та docker-compose
- Деплой на Railway або Render

## Статус проєкту

Проєкт перебуває в активній розробці. Реалізовано базову структуру та адміністративну панель. Робота над користувацькою частиною та системою замовлень триває.
